# All Believers Chapel - Church Website

## 1. Project Description
A professional, multi-page church website for All Believers Chapel (Leeds, UK). Target users are existing church members and visitors seeking a welcoming, informative online presence. Core value: convey faith, community, and accessibility with a modern, mobile-responsive design.

## 2. Page Structure
- `/` - Home (Hero, Welcome, Vision/Mission, Service Times, Ministries Preview, Scripture, Board)
- `/about` - About Us (Story, Leadership Team, Vision & Mission, Beliefs)
- `/ministries` - Ministries (Ministry categories with details)

## 3. Core Features
- [x] Sticky floating header with logo & navigation
- [x] Mobile responsive hamburger menu
- [x] Multi-page routing
- [x] Home page with full sections
- [x] About Us page
- [x] Ministries page
- [ ] Events/Calendar page (future phase)
- [ ] Sermons/Messages page (future phase)
- [ ] Contact/Give page (future phase)

## 4. Data Model Design
No backend required at this stage — all content is static.

## 5. Backend / Third-party Integration Plan
- Supabase: Not needed in Phase 1
- Shopify: Not needed
- Stripe: Potential future use for online giving
